# [Electron](https://www.electronjs.org/)

## Installation

```txt
electron-app/
├── package.json
├── main.js
└── index.html
```

```ps1
npm i -D electron@latest  # electron@beta for beta version
```

```json
{
  "name": "<app_name>",
  "version": "<SemVer>",
  "description": "",
  "main": "main.js",
  "scripts": {
    "start": "electron ."
  },
  "keywords": [],
  "author": "",
  "license": "",
  "devDependencies": {
    "electron": "<electron_version>"
  }
}
```
